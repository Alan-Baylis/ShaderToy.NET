# ShaderToy.NET
A native shadertoy-compatible GLSL playground in .NET framework Environment


## Dependecies

* NAudio Library
* SharpGL


## Supported Features

* Audio Channel (Fixed on iChannel0) MP3 File and Microphone
* Texture Channel (Fixed on iChannel1) 
* Full GLSL Support
* Load Shader from File
